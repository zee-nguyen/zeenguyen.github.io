Cosmic

What is it?

Cosmic is an image processing application in Java that was built complying with the MVC design pattern. It is an interactive program where user can load, process an image (blur, sharpen, make a mosaicstyle, generate rainbow/ checkerboard/ flags images) and save the image. User may also undo and redo an action relating to applying a filter onto an image.

All functionalities are available through buttons, menu, or the batch scripting textbox that can parse and execute a text command.

Built With

Java, Swing
