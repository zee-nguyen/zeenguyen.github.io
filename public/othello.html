Othello

What is it?

Othello is a strategy board game for two players based on the popular game Reversi. You can read more about this game here.

This project was built as an assignment in my Intensive Foundations of Computer Science course in my first semester of the ALIGN Program. It tested the students' ability to design and build a complex game from start to finish.

In my project, I also enhanced the performance of the computer player by implementing the greedy algorithm. At each turn, the computer will make the most optimal move that flips the most number of the opponent's disks.

Technoligies Used
