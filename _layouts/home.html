About -

My name is Zi Nguyen and I'm a Boston-based self-taught photographer turned web developer.

My first experience with coding was when I wanted to make some stylistic changes to my photography portfolio. Not knowing much about HTML or CSS, I embarked on several Google search adventures to find the code I needed and followed the instructions to implement them on my site. Back then, I found great power in using inline styling :)

Today, I'm a Web Developer in training. I've completed two Udemy's Web Development Bootcamp courses, learning to build web applications from start to finish. I also completed CS50X, Harvard University's introduction to Computer Science, in order to understand the code I'm writing a little better.

When I'm not in front of a computer or behind a camera, I love to explore new places with my husband, break a sweat at the gym, or attempt to make an international dish. No matter how many dishes I've tried to cook, Vietnamese food remains my strength.
