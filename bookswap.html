BookSwap

See it live | View on Github

What is it?

BookSwap is a book trading app for avid book lovers looking to trade their old books for something new.

A user can sign up for an account and start posting books that they'd like to give away. Once the books are up on the site, other users may comment to request for the books.

When a user requests for book, he/ she will receive it with postage-paid and when someone else requests for his/her book, he/ she needs to pay for the postage in advance. With media mail, each book exchange may cost a user only about $2.

Technoligies Used

HTML, CSS, JavaScript, Node.js, Express, Mongoose, MongoDB
